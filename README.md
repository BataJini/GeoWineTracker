A mobile application for wine lovers and tourists to explore Georgian wines, track tasting experiences, discover vineyards, and learn about Georgian winemaking traditions.

## Features

- **Wine Discovery**: Browse wines by grape, region, type, and fermentation method
- **Wine Details**: View comprehensive information about each wine
- **Wine Journal**: Save tasted wines with ratings and notes
- **Vineyard Map**: Explore vineyards and wine regions
- **Multi-language Support**: Georgian, English


## Future Enhancements

- Wine label scanner using the device camera
- Authentication and user profiles
- Social features (share tasting notes)
- Wine quiz and educational content
- Tour planning tools

## Tech Stack (Planned)

- **Frontend**: React Native, Expo
- **Navigation**: React Navigation
- **State Management**: React Context API or Redux
- **UI Components**: Native Base
- **Maps**: React Native Maps
- **Internationalization**: i18next
